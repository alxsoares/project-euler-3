If a = b = c = d = e = f = 0

Then we get Tau[vec(0)] AND Tau[vec(0)] = Tau[vec(0)]. So
Tau[0,0,0,0,0,0] = 0.

----------------------------------------------------------------------------

If a = b = c = d = e = f = 1

We get Tau[1,1,1,1,1,1] AND Tau[1,1,1,1,1,0] = 0

----------------------------------------------------------------------------

Tau[0,0,0,0,0,1] AND Tau[0,0,0,0,1,0] = 0

More generally:

Tau[0,0,0,d,e,f] AND Tau[0,0,d,e,f,0] = 0
