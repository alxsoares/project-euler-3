d > r (since r is the remainder) so the only possible orders are:

1. d > r > q

2. q > d > r

3. d > q > r

Case 1:

d = ar = (a^2)q ==>

    n = a^2q*q+aq = qa[1+qa] = r(1+r)
    = r^2 + r cannot be a perfect square.

Case 2:

q = ad = (a^2)r ==>

    n = ar*(a^2)r + r = r(1+(a^3)r) = r[ 1 + d^3/r^3*r ] = r [ 1 + d^3/r^2]
    = r + d^3 / r              ==>
    r(n-r) = d^3
